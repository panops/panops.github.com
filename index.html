Getting Started

The PanOps Platform is not GA. To install the latest prerelease alpha version of PanOps:

{% highlight bash %} $> # to install panops and dependencies on Ruby 1.9.3-p0, you are using rvm (http://beginrescueend.com) right? $> rvm --create use 1.9.3@panops $> gem install --pre panops Fetching: ... (100%) ... Successfully installed ... ... $> # to see what we can actually do in this release: $> panops capabilities ... $> # to learn how to use PanOps $> panops tutorial ... {% endhighlight %}
